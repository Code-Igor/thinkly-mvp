# THINKLY

Thinkly é um MVP desenvolvido para a matéria 'Fábrica de Software'.

## Funcionalidades

CRUD Tarefas: criar, ver, editar e excluir tarefa. 

## Stack

| Item                     | Valor                                                  |
|--------------------------|---------------------------------------------------------|
| Frontend     | React, JavaScript, HTML, CSS                                           |
| Build Tool   | Vite                                                                     |
| Runtime (Dev)| Node.js                                                                  |

## Possível erro de Case Sensitive

Em alguns ambientes o projeto pode apresentar erros relacionados a maiúsculas/minúsculas nos nomes dos arquivos (aconteceu comigo, corrigi o problema, mas caso aconteça novamente já está avisado).

Se isso acontecer, verifique os imports dos componentes, o nome deve corresponder exatamente ao arquivo (ex: Header.jsx ≠ header.jsx).
