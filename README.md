# THINKLY

Thinkly é um MVP desenvolvido para a matéria 'Fábrica de Software'.

## Funcionalidades

CRUD Tarefas: criar, ver, editar e excluir tarefa. 

## Stack

| Item                     | Valor                                                  |
|--------------------------|---------------------------------------------------------|
| Frontend     | React, JavaScript, HTML, CSS                                           |
| Build Tool   | Vite                                                                     |
| Runtime (Dev)| Node.js                                                                  |
